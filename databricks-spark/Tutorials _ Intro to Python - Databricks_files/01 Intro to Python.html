### **Attach** the Notebook to a **cluster**

### Working with **Spark SQL and DataFrames**
The variable **sqlContext** allows you to access a Spark SQL Context to work with Spark SQL and DataFrames.
* Python can be used to create Spark [DataFrames](http://spark.apache.org/docs/latest/sql-programming-guide.html) - a distributed collection of data organized into named columns.
* DataFrames are created by calling ``sqlContext.createDataFrame`` on an RDD of pyspark.sql.Row objects.

**Important note:** Do not create the *sqlContext* variable - it is already initialized for you.

Working with Spark SQL and DataFrames

The variable sqlContext allows you to access a Spark SQL Context to work with Spark SQL and DataFrames.

  • Python can be used to create Spark DataFrames - a distributed collection of data organized into named columns.
