# **Accessing Data**
This folder contains guides on how to access data in Databricks from various common sources.

If your data format or data source is not listed, please consult https://forums.databricks.com/ - we know that being able to access your data easily is extremely important.

### Loading data from files

These are some basic ways to import data into Databricks.

* [Data Import UI](../03 Data Sources/1 Data Import UI.html) 
  * Import simple data files using the UI.
* S3 -- in [[Python]](../03 Data Sources/2 AWS S3 - py.html) or [[Scala]](../03 Data Sources/2 AWS S3 - scala.html)
  * Read/Write data in Amazon S3.
